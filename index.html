My Projects

Belekping

An object-oriented programming-based project for a farming game where players manage fields by planting crops and raising animals, represented by cards. It's also features a GUI, save and load. Tech use - Java

Travel Chingu

A travel planer app that designed to help users organize and plan their trips efficiently. It offers a user-friendly interface to create itineraries, manage budgets, and explore recommended destinations. Tech use - Flet

Biometric Identification

Fingerprint-based biometric identification using pattern matching algorithms such as Knuth-Morris-Path(KMP), Boyer-Moore (BM), and Regular Expressions. Tech use - C#

Word Ladder's Solver

The Word Ladder game is a puzzle where players have to find a series of words that connect a starting word to an ending word, with each word in between differing by only one letter. This project implements three different search algorithms to solve the Word Ladder game: UCS (Uniform Cost Search), Greedy Best First Search, and A*. Tech use - Java and Java Swing

Diamonds Game Bot

Developed a bot for the Diamonds game, a simple game where the objective is for players to collect as many diamonds as possible on the game board. The bot was implemented using a Greedy algorithm with the primary goal of collecting the maximum number of diamonds to win the game.

WikiRace's Solver

Developed a solver for the Wikirace game, where players aim to navigate from one Wikipedia page to another using only internal links. Implemented search algorithms like Breadth-First Search (BFS) and Iterative Deepening Search (IDS). Tech use - React, Golang and D3 Graph
